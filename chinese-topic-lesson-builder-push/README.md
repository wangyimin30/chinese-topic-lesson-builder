# chinese-topic-lesson-builder

语文专题分层学案生成 skill，用于将课程内容、学生困惑、教材依据与本地题库资源整合为专题分层学案三件套：

- 学生作业版 `.docx`
- 教师评改版 `.docx`
- 课堂互动版 `.html`

核心流程为：

```text
课程内容 + 学生困惑 -> 确定专题目标 -> 资源检索 -> 素材筛选 -> 分层设计 -> 学案生成 -> 教师校验迭代
```

参考资料目前仅保留：

- `references/课程标准.pdf`
- `references/模卷库/24-26中考一模二模试题汇总3.0.html`
